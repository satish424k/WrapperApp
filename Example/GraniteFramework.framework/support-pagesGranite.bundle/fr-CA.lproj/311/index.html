Dois-je accéder à Configuration avancée pour installer mon thermostat Lyric?

Le thermostat Lyric est configuré pour contrôler les types de systèmes les plus courants, ce qui signifie que vous n'avez pas à accéder à la Configuration avancée. Toutefois, le thermostat Lyric étant conçu pour fonctionner avec une grande variété de systèmes de chauffage et de refroidissement et de produits de qualité de l'air intérieur, comme des humidificateurs pour toute la maison, des déshumidificateurs pour toute la maison ou des ventilateurs, le menu Configuration avancée vous offre plus d'options pour configurer votre équipement.
