I don't see my home Wi-Fi network listed as an available network, what should I do?

If you do not see the name of your home Wi-Fi network press the Rescan button at the bottom of the page. If your router is set to a non broadcasting mode select Add a Network and enter the Wi-Fi netork name.

Will this thermostat work with a Static IP Address Network scheme?

No, the router network needs to be setup for Dynamic Host Control Protocol (DHCP).

Does this Wi-Fi thermostat cause any interference with other wireless devices in my home?

This thermostat is Wi-Fi compliant not to cause interference.

What Wifi security standards is Lyric compatible with?

As Wi-Fi technology changes so does the security standards to protect Wi-Fi networks. The thermostats are compatible with the following security standards.

  • OPEN
  • WEP_PSK
  • WPA_TKIP_PSK
  • WPA2_AES_PSK
  • WPA2_MIXED_PSK

What Wi-Fi networks are compatible with Honeywell’s Wi-Fi thermostats?

Honeywell’s Wi-Fi thermostats are compatible with the 2.4 GHz B, G, N and Mixed Wi-Fi Networks.

How do I find the password to my router?

Check the back or bottom of the router or the operating manual for the password. If you are still unable to find the password contact the router manufacturer.

I cannot connect to my guest network. What do I need to do?

Some wireless routers have advanced features like guest networks. It is recommended that you connect your thermostat to your home network. If you must connect to your guest network, then you will need to contact the router manufacture to assist in unblocking port 443 in the guest network firewall.

Why isn’t my thermostat connecting to my Wi-Fi router even though it is very close to the thermostat?

Verify that the password entered for the Wi-Fi router is correct.

Is there a way to extend the signal strength?

Most standard routers can be set up to be a repeater. You can also purchase and install a Wi-Fi repeater.

What is the range or distance of the Wi-Fi signal in my thermostat?

The thermostat works in typical residential and light commercial applications. The range or distance of the Wi-Fi signal is determined by your router.

Lyric is not finding my in home Wi-Fi network

Lyric only operates in the 2.4GHz range.  That is why the thermostat does not see the network but the smart phone (which operates in both the 2.4 and 5.0 GHz range) sees the network just fine. Most routers will broadcast two networks, one SSID on the 5GHz radio and one SSID on the 2.4GHz radio.
